## Tujuan
Mensimulasikan tekanan RAM ketika working set aplikasi meningkat dan
menghubungkannya dengan konsep Virtual Memory.

## Metode
Simulasi dilakukan menggunakan Google Colab dengan cara mengalokasikan
memori secara bertahap hingga mendekati kapasitas RAM.

## Hasil
Penggunaan RAM meningkat secara bertahap hingga mendekati kapasitas
maksimum, yang menyebabkan perlambatan eksekusi program.

## Kesimpulan
Simulasi menunjukkan bahwa ketika working set melebihi memori fisik,
sistem akan mengalami tekanan memori yang dalam sistem nyata memicu
mekanisme Virtual Memory.
